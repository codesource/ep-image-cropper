Quelques informations:
    • Vous devez prendre des captures d'écran de tous les héros que vous voulez exporter.
    • Veillez à ce que vos héros apparaissent en entier, les héros partiellement affichés seront ignorés.
    • Les héros seront triés par couleur, étoiles et niveau.
    • Vous pouvez sélectionner plusieurs fichiers.
    • Vous cliquer sur un fichier sélectionné pour le supprimé.
    • Une fois exporté, cliquez sur les images pour les télécharger.
